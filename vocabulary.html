Vocabulary

Word Definition
Asset Anything that goes into a video game (eg. characters, objects, sound effects, maps, environments, etc)
Bug Any computer developement issue that effects the preformance of the game
Cinematics The background artistic features of the game, which the player can not control
Console The part of a computer used for communicating between the user and the character/system
Mechanics The essential functions, rules, and goals of a game. Mechanics are what make a game rewarding, entertaining, and interactive.
Physics Using real life physics to make character movment and environment to be more realistic
Tilemap A system that stores and handles tile assets for creating 2D games.
